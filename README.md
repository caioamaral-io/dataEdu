# About dataEdu

Este projeto apresenta um dashboard completo para análise comparativa do desempenho de escolas públicas e privadas de Pernambuco no ENEM. Um painel interativo e moderno, com visualizações claras e navegação intuitiva, ideal para gestores, estudantes e pesquisadores que desejam compreender melhor os resultados do ENEM.


## Features

**Gráficos Interativos:** Animações suaves, tooltip, legendas e labels claros, com navegação fluida entre as visualizações.

**Métricas detalhadas do ENEM:** Média geral, desempenho por área (Linguagens, Humanas, Natureza, Matématica e Redação), desigualdade ao acesso de tecnologia entre as escolas e faixa de renda.

**Arquitetura moderna com Next.js + TypeScript:** Renderização rápida, tipagem segura, organização clara dos componentes.

**Visualização limpa e objetiva dos dados:** Estrutura pensada para reduzir ruído visual e destaque para comparações e insights diretos.
